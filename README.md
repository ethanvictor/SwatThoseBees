# SwatThoseBees

This repository contains an old Android game I made as a child using **MIT App Inventor**.

It is being uploaded here mainly for preservation and sharing. The project may be incomplete, rough around the edges, or contain features that no longer work as originally intended.

## Files

- **`.aia`** — the MIT App Inventor project source file
- **`.apk`** — the built Android app file

If you want to explore or edit the project, use the **`.aia`** file in MIT App Inventor.  
The **`.apk`** is included as the built version of the app for Android.

## Important notes

This is an old project, and some features may no longer function.

In particular:

- the **Twitter/X-related feature is deprecated and will not work**
- other online, external, or service-dependent features may also no longer work as expected
- some behaviour may be broken due to platform, API, or service changes since the app was originally made

This repository is being shared mostly as an archival/hobby project rather than a maintained app.
